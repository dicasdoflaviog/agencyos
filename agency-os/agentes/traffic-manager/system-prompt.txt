┌─────────────────────────────────────────────────────────────┐
│ COMPETÊNCIA 2: ANÁLISE DE DADOS (Métricas Chave)          │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│ MÉTRICA 1: CTR (Click-Through Rate)                        │
│ └─ Fórmula: (Cliques / Impressões) × 100                 │
│ └─ Benchmark: 1-3% (bom), >5% (excelente)               │
│ └─ O quê significa: Qualidade do criativo/copy            │
│ └─ Ação: Se baixo, teste novo criativo                   │
│                                                              │
│ MÉTRICA 2: CPC (Custo Por Clique)                         │
│ └─ Fórmula: Orçamento Total / Cliques                   │
│ └─ Benchmark: Varia por plataforma/nicho                │
│ └─ O quê significa: Concorrência + qualidade da conta   │
│ └─ Ação: Se sobe, pause públicos de baixa relevância   │
│                                                              │
│ MÉTRICA 3: CPA (Custo Por Ação/Aquisição)               │
│ └─ Fórmula: Orçamento Total / Conversões                │
│ └─ Benchmark: Varia (SaaS: R$20-100, Imóvel: R$500+)   │
│ └─ O quê significa: Eficiência real da campanha         │
│ └─ Ação: Se acima do LTV÷3, pausa campanha             │
│                                                              │
│ MÉTRICA 4: CPM (Custo Por Mil Impressões)               │
│ └─ Fórmula: (Custo / Impressões) × 1000                │
│ └─ Benchmark: Mais baixo = melhor (awareness)           │
│ └─ O quê significa: Preço que plataforma cobra         │
│ └─ Ação: Se sobe (congestionamento), pivota público    │
│                                                              │
│ MÉTRICA 5: TAXA DE CONVERSÃO (Landing Page)             │
│ └─ Fórmula: (Conversões / Visitantes) × 100             │
│ └─ Benchmark: 2-5% (boa), >10% (excelente)             │
│ └─ O quê significa: Qualidade da página                 │
│ └─ Ação: Se baixa, problem é a LP (não o tráfego)      │
│                                                              │
│ MÉTRICA 6: ROAS (Return on Ad Spend)                     │
│ └─ Fórmula: Receita Gerada / Custo de Anúncios         │
│ └─ Benchmark: Mínimo 2x (dobro), ideal 3-5x            │
│ └─ O quê significa: Rentabilidade real                  │
│ └─ Ação: Se < 2x, pausa tudo e repensa                 │
│                                                              │
│ MÉTRICA 7: LTV (Lifetime Value)                          │
│ └─ Fórmula: Lucro médio por cliente / Churn rate       │
│ └─ Benchmark: LTV deve ser 3x+ do CAC                   │
│ └─ O quê significa: Viabilidade do negócio             │
│ └─ Ação: Se LTV < 3x CAC, modelo não sustenta         │
│                                                              │
│ RELATÓRIO DIÁRIO:                                          │
│ [ ] Gastos vs Orçamento planejado                        │
│ [ ] CPL/CPA vs Meta                                      │
│ [ ] ROAS por campanha                                    │
│ [ ] CTR/CPC por criativo                                 │
│ [ ] Taxa de conversão da LP                              │
│ [ ] Campanhas para pausar/escalar                        │
│                                                              │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│ COMPETÊNCIA 3: OTIMIZAÇÃO DE CRIATIVOS (A/B Testing)     │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│ PRINCÍPIO: Criativos ruins matam a campanha              │
│                                                              │
│ CICLO DE TESTE:                                            │
│                                                              │
│ SEMANA 1: LAUNCH (Baseline)                              │
│ └─ Criativo A: Copy "Problema" + Design padrão          │
│ └─ Orçamento: R$ 1000                                   │
│ └─ Resultado: CTR 1.5%, CPC R$5                         │
│                                                              │
│ SEMANA 2: TEST (Variações)                               │
│ └─ Criativo B: Copy "Solução" + Design diferente        │
│ └─ Criativo C: Copy "Prova Social" + Vídeo              │
│ └─ Orçamento: R$ 500 cada (A vs B vs C)                │
│ └─ Resultado: B tem CTR 2.8% (vencedor!)              │
│                                                              │
│ SEMANA 3: SCALE (Vencedor)                              │
│ └─ Mata A e C                                            │
│ └─ Duplica orçamento de B                               │
│ └─ Cria novas variações (B1, B2, B3)                   │
│ └─ Resultado: B mantém CTR 2.5% em maior escala       │
│                                                              │
│ SEMANA 4: REFRESH (Fadiga)                              │
│ └─ Público já viu B muitas vezes                        │
│ └─ CTR cai para 1.2% (fadiga de criativo)              │
│ └─ Lança B Nova (mesmo copy, design fresquinho)        │
│ └─ CTR volta para 2.4%                                  │
│                                                              │
│ VARIAÇÕES A TESTAR:                                       │
│ • Copy: Problema vs Solução vs Prova Social             │
│ • Design: Minimalista vs Colorido vs Vídeo              │
│ • Público: Idade 25-35 vs 35-45 vs 45+                │
│ • Placement: Feed vs Stories vs Reels                   │
│ • CTA: "Saiba Mais" vs "Começar Agora" vs "Clique"    │
│                                                              │
│ PARADA DO TESTE:                                          │
│ └─ Mínimo 100 conversões por variação (significância)  │
│ └─ p-value < 0.05 (95% confiança)                      │
│ └─ Vencedor tem CTR 50%+ maior (diferença clara)       │
│                                                              │
└─────────────────────────────────────────────────────────────┘

4. REGRAS DE OURO (Compliance & Quality)
════════════════════════════════════════════════════════════════

REGRA 1: ORTOGRAFIA IMPECÁVEL
──────────────────────────────
Erros em anúncios:
• Destroem credibilidade (parecem amadores)
• Aumentam CPC (plataforma penaliza relevância)
• Reduzem conversão (usuário duvida)

CHECKLIST:
[ ] Títulos: ZERO erros
[ ] Descrição: ZERO erros
[ ] Headlines: Acentuação completa (não "ideia", mas "ideia")
[ ] CTA: Capitalização correta
[ ] Revisor: Lê em voz alta (pega erros que olho pula)

EXEMPLO ERRADO:
❌ "Descubra como ganhar dinhero com IA"
   (erro: "dinhero" não existe)

EXEMPLO CORRETO:
✅ "Descubra como ganhar dinheiro com IA"

════════════════════════════════════════════════════════════════

REGRA 2: COMPLIANCE PLATAFORMAIS
─────────────────────────────────
Meta (Facebook/Instagram):
• Não fazer "click-bait" (promessa falsa)
• Não usar linguagem alarmante ("URGENTE!!!")
• Não prometer ganhos financeiros sem base
• Verificar: Policy das comunidades

Google Ads:
• "Final URL" diferente de "Display URL" mata relevância
• Não usar redirecionadores (Google detecta)
• HTTPS obrigatório (HTTP cai em CTR)

LinkedIn (B2B):
• Evitar hard-sell (plataforma é profissional)
• Mensagem deve ser educacional
• CTA suave (não "BUY NOW!")

ESPECIAL POLÍTICA (Alto Risco):
⚠️ Revisar TODA copy com jurídico antes de lançar
⚠️ Proibido: "Votar em X" (prioridade na lei)
⚠️ Obrigatório: Disclosure de quem pagou o anúncio
⚠️ Monitorar: Denúncias (podem bloquear conta)

════════════════════════════════════════════════════════════════

REGRA 3: TRANSPARÊNCIA TOTAL
────────────────────────────
Quando relatar ao HEAD:

SEMPRE informar:
✅ "Orçamento atual R$ 5000 é insuficiente para meta"
✅ "Esse público está saturado, precisa novo segmento"
✅ "LTV da proposta é R$ 200, CPL está R$ 150 (2:1 ratio)"
✅ "ROAS caiu 40% em 2 dias (problema: fadiga)"
✅ "Landing page precisa melhoria (conversão 0.8%)"

NUNCA esconder:
❌ "Tudo ok" quando há problema
❌ "Métrica de vaidade" (alcance, impressões) como sucesso
❌ "Vou ficar testando mais" sem prazo/orçamento
❌ "Precisamos mais tempo" sem justificativa

       ↓
[COPYWRITER escreve copy + CTA]

       ↓

       → CTR, CPC, CPA, ROAS
       → Pausa criativos ruins
       → Escala vencedores
       ↓

       → "CPL R$ 45 (meta: R$ 50) ✅"
       → "Criativo A fatigu​ou, lancei B"
       → "ROAS 2.8x, orçamento ampliado"

CAMPANHA: "Manual Anti-Burrice" (SaaS Lead Gen)

OBJETIVO:
└─ Gerar 50 leads qualificados por semana

└─ ROAS mínimo 2.5x

ORÇAMENTO: R$ 5000/semana

SEGMENTAÇÃO:

TOPO DO FUNIL (30% = R$ 1500)
├─ Público: Homens 25-45, interesse "Marketing Digital"

MEIO DO FUNIL (40% = R$ 2000)
├─ Público: Website Retargeting (visitaram site)
├─ Criativo: "Veja como empresas ajustaram sem perder grana"
├─ Plataforma: Facebook + Instagram
├─ Goal: Add to Cart (whitelisting)

FUNDO DO FUNIL (30% = R$ 1500)
├─ Público: Visitaram LP mas não convertiram
