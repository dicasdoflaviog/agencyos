Cada rede social é DIFERENTE. Mesma arte, estratégia diferente.

INSTAGRAM / FACEBOOK (Foco em Engajamento)
──────────────────────────────────────────
Algoritmo: Premia comentários, salvamentos, compartilhamentos
          (não só likes)

SUA OTIMIZAÇÃO:
├─ Legenda: 150-250 caracteres (não "paredes de texto")
├─ Primeiro 125 caracteres = GANCHO (para aparecer antes do "mais")
├─ CTA claro: "Comenta aqui", "Salva isso", "Compartilha com alguém"
├─ Hashtags: 20-30 relevantes (no final ou 1º comentário)
│  └─ Padrão: 10 grandes (>1M posts), 10 médias (100k-1M), 10 pequenas (<100k)
│  └─ Exemplo: #IA #inteligenciaartificial (grandes)
│             #AIforsmallbusiness #estrategiaantifrágil (médias)
│             #dicasdoflaviog #manualantiburrice (pequenas/branded)
├─ Emojis: Máx 3-5 (para destaque, não poluição)
└─ Agendamento: Via Meta Business Suite (Instagram + Facebook juntos)

MELHOR HORA INSTAGRAM:
└─ Depende de seu público, mas TESTE:
   ├─ Terça-quinta 17h-19h (após trabalho)
   ├─ Sábado 10h-12h (leisure time)
   └─ Você valida o que funciona com seu público

LINKEDIN (Foco em Autoridade & Profissionalismo)
─────────────────────────────────────────────────
Algoritmo: Premia comentários (mais que likes), coerência de voz,
           conexão com rede

SUA OTIMIZAÇÃO:
├─ Legenda: 1500-2000 caracteres (LinkedIn ADORA textos longos)
├─ Estrutura: Abertura forte → Contexto → Insights → Chamada
├─ Parágrafo 1: Frase de impacto (gancho)
├─ Parágrafos 2-4: Desenvolvimento com quebras visuais (Enter entre parágrafos)
├─ Parágrafo final: CTA claro ("Me responde:" / "Comente aqui:")
├─ Hashtags: 5-10 tags profissionais (#Estratégia #IA #Empreendedorismo)
├─ Menções: Marca clientes, parceiros (aumenta reach)
└─ Agendamento: Post manual (LinkedIn adora organic, não react bem com scheduler)

MELHOR HORA LINKEDIN:
└─ Terça-quinta 8h-10h (início do dia)
   ou 17h-19h (final do dia, planning mode)

TIKTOK / YOUTUBE SHORTS (Foco em SEO & Algoritmo)
──────────────────────────────────────────────────
Algoritmo: Premia watch time (quanto tempo assiste),
           completion rate (assistiu até o fim?)

SUA OTIMIZAÇÃO:
├─ Título: Máx 60 caracteres, SEO pesado
│  ├─ Padrão: [PALAVRA-CHAVE] + [PROMPT] + [CURIOSIDADE]
│  ├─ ❌ Errado: "Vídeo novo"
│  ├─ ✅ Certo: "IA para freelancers — 5 usos que ninguém te conta"
├─ Descrição: 100-150 caracteres com palavra-chave 2x
│  └─ Primeira linha: Resumo do vídeo (aparece antes de expandir)
├─ Tags: 10-15 palavras-chave (YouTube adora tags precisas)
├─ Hashtags: #FYP #Para Você (TikTok), #Shorts (YouTube)
├─ Thumbnail: Imagem custom (não frame do vídeo)
│  └─ Contraste alto, expressão clara, texto <5 palavras
├─ Primeira cena: CRUCIAL (3 primeiros segundos definem se continue assistindo)
└─ Duração: 15-60 segundos (TikTok/Shorts) — não demore

MELHOR HORA TIKTOK/SHORTS:
└─ 17h-22h (quando pessoas estão no phone)

PINTEREST (Foco em Tráfego & Inseminação)
──────────────────────────────────────────
Algoritmo: Premia salvamentos (muito mais que repins)
           Cliques para fora (tráfego para LP)

SUA OTIMIZAÇÃO:
├─ Descrição: 300-500 caracteres com SEO
│  ├─ Palavra-chave na frente: "IA para freelancer: como ganhar 10h/semana"
│  └─ Descrever benefício claramente
├─ Hashtags: 5-10 hashtags (Pinterest as trata como palavras-chave)
├─ Pin: Imagem 1000 × 1500 px (ratio 2:3)
│  ├─ Contraste alto
│  ├─ Texto bold (máx 20 palavras)
│  └─ Cores saturadas (Pinterest adora saturation)
├─ Link: Sempre incluir link funcional (para LP, blog, etc)
├─ Boards: Organizar em 3-5 boards temáticos
└─ Agendamento: Via Pinterest Ads ou Repinagem manual

MELHOR HORA PINTEREST:
└─ Qualquer hora (Pinterest não é time-sensitive como Instagram)
   Mas segunda/terça/quarta tendem a funcionar melhor

WHATSAPP / TELEGRAM (Foco em Conversão)
─────────────────────────────────────────
NÃO é "publicação" como outras redes. É "disparo" para leads qualificados.

SUA OTIMIZAÇÃO:
├─ Script de mensagem: 50-100 caracteres máx (mobile-first)
├─ Estrutura: Gancho → Proposta → Urgência → Link
├─ Links: SEMPRE com UTM rastreável
│  └─ Padrão: link.base.com?utm_source=whatsapp&utm_campaign=antiburrice
├─ Imagem: Card visual (1080 × 1350 px) ou GIF
├─ Timing: Disparo em horário que abrem (depende grupo/lista)
└─ Ferramenta: Disparador automático (Zapier, Make) ou manual

EXEMPLO WHATSAPP:
"📱 Sua rotina em metade do tempo?
IA sem estrutura FALHA.
Aprenda a lógica por trás → link.com/antiburrice"

├─ Repetir 2-3x ao longo do texto (natural, não forçado)
└─ Exemplo: "IA para freelancer"

   ├─ Legenda: "Se você é freelancer e usa IA, precisa ler isso..."

   ├─ Upload de Shorts

   ├─ Agendamento de pins
   ├─ Repinagem estratégica

CHECKLIST DE SEGURANÇA (O OBRIGATÓRIO)
───────────────────────────────────────
Antes de clicar "publicar", RESPONDA TODAS as perguntas:

LINKS:
□ Link CTA funciona? (Testei clicando?)
□ Link tem UTM correto?
   └─ Padrão: ?utm_source=[rede]&utm_campaign=[campanha]&utm_medium=social
□ Link não está quebrado/encurtado mal?
□ Landing page carrega rápido? (user vai clicar se lento = bounce)

IMAGENS/VÍDEOS:
□ Imagem tem dimensão correta para plataforma?
□ Qualidade alta? (pixelada = rejeição)
□ Alt Text descritivo em TODAS as imagens?
   └─ Padrão: "[Descrição simples da imagem] — Manual Anti-Burrice"
□ Nenhum criativo bloqueado por política de plataforma?
   └─ Evitar: Linguagem política agressiva, fake news, NSFW

TEXTO:
□ Português impecável? (última chance de salvar!)
   ├─ Acentuação correta?
   ├─ Concordância ok?
   └─ Pontuação coerente?
□ Tom "Antiguru" preservado? (sem 🚀🔥💣, sem promessas milagrosas)
□ CTA claro e coerente com rede?
□ Primeira linha tem gancho (legível em preview)?

HASHTAGS:
□ Hashtags relevantes para nicho? (não spam)
□ Quantidade certa por plataforma?
   ├─ Instagram: 20-30
   ├─ LinkedIn: 5-10
   ├─ TikTok: 5-10
   └─ Pinterest: 5-10
□ Spelling correto (uma letra errada = hashtag inútil)?

AGENDAMENTO:
□ Horário otimizado para público alvo?
□ Data faz sentido? (não postar fim de semana se foco é B2B)
□ Não está conflitando com outro post?
   └─ Padrão: Máx 1 post/dia por rede

MARCA:
□ Logo/assinatura coerente?
□ Cores alinhadas com brand?
□ Fonte/tipografia é marca?

PRONTO PARA PUBLICAR?
□ Checklist 100% completo?
□ HEAD aprovou conteúdo antes disso? (você valida, não aprova conceito)
□ Documentei em Drive/Planilha?
