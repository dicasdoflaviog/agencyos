AGENCY
OS

Painel central de operação da eugência. Gerencia clientes, aciona agentes, aprova criativos, conecta Instagram, WhatsApp, ManyChat, Meta Ads e Figma. Portal de aprovação para o cliente. Dashboard financeiro. CMS para blog e sites.

Stack Next.js 14 + Supabase + Vercel
Clientes gerenciados dicasdoflaviog + QG Digital + terceiros
Acionamento agentes Misto — manual p/ clientes, auto p/ marca
Aprovação cliente Portal com link seguro
Idiomas PT-BR agora → EN futuro (white-label)
Potencial SaaS Sim — se funcionar bem, vira produto
Custo MVP R$ 0

Stack técnica completa
Front-end
Next.js 14 (App Router)
React + TypeScript. SSR para portal do cliente. Rotas protegidas por role. Mesmo padrão do QG Digital.
Tailwind CSS + shadcn/ui
Design system da dicasdoflaviog aplicado via CSS variables. Componentes base customizados.
Vercel
Deploy automático a cada push. HTTPS incluso. Free tier suficiente para o MVP inteiro.
Back-end / API
Next.js API Routes
Endpoints para acionar agentes, processar webhooks ManyChat/Instagram, chamar Playwright.
Playwright (server-side)
Renderiza HTML dos slides → exporta PNG 1080×1350. Roda em API Route no Vercel.
Anthropic API
Integração com os 5 agentes. Passa contexto visual (imagens, arquivos, design system) direto pela API.
Banco / Storage
Supabase (Postgres)
Banco relacional com RLS por role. Free tier: 500MB. Realtime subscriptions para status de jobs.
Supabase Storage
Upload de logos, PDFs, design systems, PNGs gerados. Organizado por client_id. CDN incluso.
Supabase Auth
Autenticação de admin e colaboradores. Tokens de acesso para portal do cliente sem login.
IA + Automação
Claude claude-sonnet-4-6 (Anthropic API)
Modelo principal de todos os agentes. Suporte nativo a imagem, PDF e arquivos no contexto.
Figma REST API
Agente Designer lê templates em tempo real. Design tokens sempre atualizados da fonte.
Vercel Cron Jobs
Agendamento automático. Toda segunda aciona o agente para gerar conteúdo da dicasdoflaviog.
