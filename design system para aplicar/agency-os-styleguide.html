Animações & Micro-interações

Tokens de easing/duração e todas as animações do sistema — com demos ao vivo

Curvas de Easing

--ease-spring
cubic-bezier(0.34, 1.56, 0.64, 1)
Efeito "mola" — ultrapassa levemente o destino. Ideal para botões, badges e elementos que surgem.
--ease-smooth
cubic-bezier(0.4, 0, 0.2, 1)
Suave e natural. Padrão para a maioria das transições: cores, bordas, sombras.
--ease-decelerate
cubic-bezier(0, 0, 0.2, 1)
Entra rápido e desacelera. Para elementos que entram na tela: modais, sidepanels.
--ease-sharp
cubic-bezier(0.4, 0, 1, 1)
Aceleração rápida. Para elementos que saem da tela: fechar toasts, dispensar.

Como usar no CSS

/* Entrada de elemento na tela */ .meu-card { animation: fade-up 300ms var(--ease-decelerate) both; } /* Delay em cascata */ .card:nth-child(1) { animation-delay: 0ms; } .card:nth-child(2) { animation-delay: 60ms; } .card:nth-child(3) { animation-delay: 120ms; } /* Micro-interação de hover */ .card-interactive { transition: transform var(--dur-base) var(--ease-smooth), box-shadow var(--dur-base) var(--ease-smooth); } .card-interactive:hover { transform: translateY(-2px); box-shadow: var(--shadow-md); } /* Botão com spring no :active */ .btn:active { transform: scale(0.96); transition-duration: var(--dur-fast); transition-timing-function: var(--ease-spring); }
