## O QUE É A KEYWORDSBAR

A KeywordsBar é a barra no rodapé inferior de cada slide que exibe
3 palavras-chave relacionadas ao tema do conteúdo.

Ela aparece abaixo da linha divisória e acima do final do frame.
Não é o rodapé da marca — é uma camada de contexto temático.

Função: identificar de forma visual e rápida o tema do post,
reforçar SEO semântico e criar consistência de identidade editorial.


## ONDE APLICAR

Template A — Capa Editorial    : SEMPRE (slide 1)
Template C — Conteúdo / Lista  : SEMPRE (slides internos)
Template D — CTA / Encerramento: OPCIONAL (use se houver espaço)
Template B — Foto + Headline   : NUNCA (conflita visualmente com foto)
Stories                        : NUNCA (formato não comporta)


## REGRAS DE USO

1. SEMPRE 3 keywords — nunca 2, nunca 4.
2. Todas em MAIÚSCULO e sem acento quando possível (ex: IA, não INTELIGÊNCIA).
3. Máximo 20 caracteres por keyword.
4. Nunca repetir a mesma palavra do headline do slide.
5. Usar a tabela abaixo como referência — não inventar.

## COMO O AGENTE DESIGNER APLICA

No brief recebido do Estrategista, o campo "pillar" e "keywords"
já virão preenchidos. Exemplo:

  pillar: "IA Aplicada"
  keywords: ["IA", "AUTOMAÇÃO", "PRODUTIVIDADE"]

O Agente Designer usa esses valores para preencher o componente
KeywordsBar no Figma ou no HTML gerado pelo Playwright.

Em HTML, a KeywordsBar é renderizada assim:

  <div class="keywords-bar">
    <span>IA</span>
    <div class="sep"></div>
    <span>AUTOMAÇÃO</span>
    <div class="sep"></div>
    <span>PRODUTIVIDADE</span>
  </div>

CSS base:
  .keywords-bar {
    display: flex;
    align-items: center;
    justify-content: space-between;
    padding: 16px 72px;
    border-top: 1px solid rgba(255,255,255,0.10); /* dark mode */
    font-family: 'DM Sans', sans-serif;
    font-weight: 700;
    font-size: 13px;
    letter-spacing: 0.06em;
    color: rgba(245,242,236,0.45); /* dark mode */
  }
  .keywords-bar .sep {
    width: 1px;
    height: 18px;
    background: rgba(255,255,255,0.15);
  }
  /* Light mode overrides */
  .keywords-bar.light {
    border-top-color: #E8E5DF;
    color: rgba(13,13,13,0.40);
  }
  .keywords-bar.light .sep {
    background: #E8E5DF;
  }

## QUANDO CRIAR KEYWORDS CUSTOMIZADAS

Se o conteúdo não se encaixar em nenhuma categoria da tabela,
o Agente Copywriter deve sugerir 3 keywords no brief.
O Agente Designer NUNCA inventa keywords sem instrução do brief.
