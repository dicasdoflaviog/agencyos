COMO VOCÊ USA TUDO ISSO DIARIAMENTE
════════════════════════════════════════════════════════════════════════════════

⏰ 08h00 - CHECK DIÁRIO (5 min)
────────────────────────────
1. Abre: Drive → 04_PERFORMANCE/Daily_Dashboard
2. Verifica:
   ├─ Aba "Leads": Quantos novos? Estão bons?
   ├─ Aba "Performance": CPL está OK?
   └─ Aba "ROI": Está no alvo de 5.5x?
3. Se algo está errado → Toma ação (reduz orçamento, muda copy)
4. Documenta em: Weekly_Reports (1x/semana)

📧 Quando recebe ALERTA (CPL > R$ 200)
───────────────────────────────────
1. Recebe email/notificação Make
2. Avalia a situação
3. Toma ação rápida (15 min):
   ├─ Reduz orçamento em 15-20%
   ├─ Pausa público com pior performance
   └─ Acelera teste de novo copy
4. Monitora resultado nas próximas 12-24h

📊 1x/Semana - RELATÓRIO (30 min)
─────────────────────────────────
1. Abre Dashboard
2. Cria resumo: "Semana de [X-Y]"
   ├─ Total de leads: X
   ├─ CPL médio: R$ Y
   ├─ Melhores dias: [dias]
   └─ Próximas ações: [lista]
3. Salva em: Drive → 04_PERFORMANCE/Weekly_Reports
4. Compartilha com cliente (resumido)

📈 Mensalmente - GRANDE RELATÓRIO (2 horas)
──────────────────────────────────────────
Estrategista compila tudo:
├─ Total de leads acumulado
├─ Custo total gasto
├─ ROI alcançado até agora
├─ Quais temas geraram mais leads
├─ Melhor dia/hora para publicar
├─ Recomendações para próximo mês
└─ Cria apresentação ao cliente (Google Slides)

CONCLUSÃO: O QUE VOCÊ GANHOU
════════════════════════════════════════════════════════════════════════════════

Antes (sem sistema):
├─ Entrada manual de dados (3h/semana)
├─ Sem visibilidade em tempo real
├─ Não sabe qual copy funciona
├─ Não sabe qual horário é melhor
└─ Sem rastreio de ROI exato

Depois (com sistema):
├─ ✅ Automação 99% (apenas lê resultados)
├─ ✅ Dados em tempo real (a cada 15 min)
├─ ✅ Sabe exatamente qual copy gera leads
├─ ✅ Sabe qual horário publicar
├─ ✅ ROI rastreado com precisão de centavos
├─ ✅ Escalável para 10+ clientes
└─ ✅ Tempo para focar em estratégia (não admin)
