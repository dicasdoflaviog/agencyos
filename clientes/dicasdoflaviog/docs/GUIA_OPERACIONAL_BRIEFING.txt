PARTE 1: ESTRUTURA DE BRIEFING EXPLICADA
═══════════════════════════════════════════════════════════════════════════════

O briefing é o COMBUSTÍVEL de todo seu squad. Sem ele, ninguém sabe o que fazer.

NÍVEIS DE BRIEFING (você vai usar 3):

┌─────────────────────────────────────────────────────────────────────────────┐
│ NÍVEL 1: BRIEFING DE CLIENTE (Fase 01)                                     │
│ Responsável: ESTRATEGISTA                                                    │
│ Quando: Na primeira conversa com o cliente                                  │
│ Documentação: NotebookLM (cliente vê tudo)                                  │
└─────────────────────────────────────────────────────────────────────────────┘

O que contém:
├─ Contexto: "Quem é o cliente? O que faz? Qual é o negócio?"
├─ Público: "Quem é o cliente ideal? Como ele pensa? Onde está?"
├─ Mercado: "Qual é a oportunidade? Quem são os competidores?"
├─ Dor Principal: "Qual é o maior problema que o cliente quer resolver?"
├─ Objetivo Semestral: "Qual é a meta? (leads, vendas, visibilidade?)"
├─ Orçamento: "Quanto vai investir em marketing?"
├─ Histórico: "Tentou algo antes? O que funcionou/não funcionou?"
└─ Preferências: "Qual é o tom de voz? Quais plataformas quer usar?"

Exemplo estrutura documento:
```
BRIEFING DE CLIENTE — Manual Anti-Burrice
═════════════════════════════════════════

CONTEXTO
├─ Cliente: Flávio G. (dicasdoflaviog)
├─ Negócio: Educação em IA + Automação
├─ Público: Empreendedores, freelancers, donos de agência
└─ Estágio: Crescimento (já tem autoridade, quer escalar)

PÚBLICO-ALVO
├─ Quem: Profissionais com 2-10 anos de experiência
├─ Dor: Gastam 30-50h/mês em tarefas repetitivas
├─ Plataformas que usam: Instagram, LinkedIn, WhatsApp
└─ Nível de consciência: Sabem que problema existe, não sabem solução

MERCADO
├─ Oportunidade: IA em alta, mas cursos genéricos dominam
├─ Diferencial: Ninguém ensina LÓGICA, só prompt
└─ Competidores: Cursos baratos (R$ 17-47) sem credibilidade

OBJETIVO
├─ Meta: 1.500 leads qualificados em 6 meses
├─ Orçamento: R$ 10.000/mês
└─ Resultado esperado: R$ 120k em receita (1.500 × R$ 80)

HISTÓRICO
├─ Tentou: Instagram orgânico (funcionou pouco)
├─ Não tentou: Google Ads, LinkedIn Ads, Email sequence
└─ Histórico de vendas: Conversão ~10% de leads interessados
```

┌─────────────────────────────────────────────────────────────────────────────┐
│ NÍVEL 2: BRIEFING ESTRATÉGICO (Fase 02)                                    │
│ Responsável: ESTRATEGISTA (coordena squad)                                  │
│ Quando: Após aprovação do cliente                                           │
│ Documentação: NotebookLM (detalhado para squad)                             │
└─────────────────────────────────────────────────────────────────────────────┘

O que contém (expande o Nível 1):
├─ SWOT (Strengths, Weaknesses, Opportunities, Threats)
├─ Funil de Vendas (Topo → Meio → Fundo com metas)
├─ Calendário de 6 meses (temas por semana/mês)
├─ Ângulo Fora da Curva (a provocação central)
├─ Distribuição de orçamento (qual % pra cada rede?)
├─ Público-alvo detalhado (personas, padrões de comportamento)
├─ KPIs por fase (impressões, cliques, conversão, CPL, ROAS)
└─ Aprovações e Assinaturas (contrato SOW)

Exemplo estrutura:
```
BRIEFING ESTRATÉGICO — Manual Anti-Burrice
═══════════════════════════════════════════

SWOT ANÁLISE
├─ Strengths: Expertise claro, comunidade engajada, conteúdo único
├─ Weaknesses: Marca jovem, baixo volume de seguidores
├─ Opportunities: IA em alta, demanda por educação prática
└─ Threats: Cursos baratos, falta de prova social inicial

FUNIL SEMESTRAL
├─ TOPO (Awareness): 100k impressões/mês
│  ├─ Conteúdo: "5 padrões de IA ninguém te conta"
│  ├─ Rede: Instagram, TikTok, Pinterest
│  └─ CTA: "Salva", "Compartilha"
│
├─ MEIO (Consideration): 10k cliques/mês
│  ├─ Conteúdo: "Como ganhei 12h/semana"
│  ├─ Rede: LinkedIn, Email, Anúncios retargetados
│  └─ CTA: "Baixa este recurso"
│
└─ FUNDO (Conversion): 1.500 leads/mês
   ├─ Conteúdo: "Manual Anti-Burrice + 7 dias email"
   ├─ Rede: Landing Page, WhatsApp
   └─ CTA: "Compra agora", "Entra no grupo"

CALENDÁRIO (Mês 1)
├─ Semana 1: "5 padrões de IA" (IG/TK), Email sequence setup
├─ Semana 2: "Por que prompt falha" (IG/TK/LI), A/B test anúncios
├─ Semana 3: "Caso de uso real" (IG/TK), Retargeting iniciado
└─ Semana 4: "Oferta + Urgência" (todas redes), Conversão push

DISTRIBUIÇÃO ORÇAMENTO
├─ Instagram/Facebook: 40% (R$ 4.000)
├─ Google Ads: 35% (R$ 3.500)
├─ LinkedIn: 25% (R$ 2.500)
└─ Total: R$ 10.000/mês

KPIs MÊS 1
├─ Impressões: 100k
├─ CTR (Click-Through Rate): 3%+ (bom)
├─ Leads: 250+
├─ CPL (Custo Por Lead): R$ 40
└─ ROAS: 2x minimum
```

┌─────────────────────────────────────────────────────────────────────────────┐
│ NÍVEL 3: BRIEFING POR PROJETO/TEMA (Semanal)                               │
│ Responsável: ESTRATEGISTA (para cada conteúdo novo)                         │
│ Quando: Antes de Copywriter/Designer trabalhar                             │
│ Documentação: Google Doc ou Notion (squad acessa rápido)                    │
└─────────────────────────────────────────────────────────────────────────────┘

O que contém (mini-briefing):
├─ Objetivo do Projeto: "Venda, Captação, Autoridade?"
├─ Público e Nível de Consciência: "Quem, onde, qual estágio?"
├─ Ângulo Fora da Curva: "Qual é a provocação?"
├─ Estrutura Narrativa: "[GANCHO] → [PROBLEMA] → [SOLUÇÃO] → [CTA]"
├─ Diretriz para Squad: "Copy: foque em X. Design: use Y. Tráfego: teste Z."
├─ Potencial de Escala: "Alto/Médio/Baixo (para Traffic Manager)"
└─ Próximos Passos: "Quem faz o quê? Quando? Prazo?"

Exemplo estrutura:
```
BRIEFING DE PROJETO — "IA Para Freelancer"
═════════════════════════════════════════

OBJETIVO
Captar 50 leads de freelancers interessados em automação com IA.

PÚBLICO
├─ Quem: Freelancers de 2-8 anos de experiência
├─ Dor: Gastam 3h/dia em tarefas manuais (responder email, editar imagem)
├─ Consciência: Sabem que problema existe, acham IA cara/complexa
└─ Plataforma: Ativo em Instagram, TikTok, WhatsApp

ÂNGULO
"IA não é cara. Você que tá gastando 2h em coisa que máquina faz em 2 minutos."

NARRATIVA
[GANCHO] Quanto você gasta/dia em tarefas idiotas?
[PROBLEMA] Acredita que IA é cara porque não sabe estruturar
[INSIGHT] IA + estrutura lógica = 2h economizadas/dia (prova)
[CTA] "Quer aprender a estrutura?" → Manual Anti-Burrice

DIRETRIZES
├─ Copy: Ton antifrágil, zero hype. Foca em "economia de tempo" (número)
├─ Design: Minimalista, preto + amarelo, destaque em CTA
├─ Tráfego: Teste público "Freelancers Brasil". Potencial escala: ALTO
└─ Prazo: Entregar tudo até sexta (publicação segunda)

PRÓXIMOS PASSOS
├─ Copywriter: Escreve legenda + 3 variações de anúncio (hoje)
├─ Designer: Cria 5 slides (até amanhã)
├─ Traffic Manager: Setup de audiência (até quarta)
└─ Publicador: Validação + agendamento (até sexta)
```

PARTE 2: COMO COLETAR DADOS (Entrevista com Cliente)
═══════════════════════════════════════════════════════════════════════════════

Quando um NOVO CLIENTE chega, o ESTRATEGISTA faz uma entrevista estruturada.

ROTEIRO DE ENTREVISTA (60-90 minutos)
─────────────────────────────────────

┌─────────────────────────────────────────────────────────────────────────────┐
│ BLOCO 1: CONTEXTO E NEGÓCIO (15 min)                                       │
└─────────────────────────────────────────────────────────────────────────────┘

Perguntas:
├─ "Conte-me sobre seu negócio. O que exatamente você faz?"
│  └─ Objetivo: Entender o modelo de negócio
│
├─ "Há quanto tempo está no mercado?"
│  └─ Objetivo: Entender estágio (startup, crescimento, maduro)
│
├─ "Qual é seu maior diferencial comparado à concorrência?"
│  └─ Objetivo: Identificar ponto único (pode virar ângulo)
│
└─ "Qual é a maior meta/objetivo para os próximos 6-12 meses?"
   └─ Objetivo: Entender ambição (vai afetar orçamento e estratégia)

┌─────────────────────────────────────────────────────────────────────────────┐
│ BLOCO 2: PÚBLICO-ALVO (15 min)                                             │
└─────────────────────────────────────────────────────────────────────────────┘

Perguntas:
├─ "Descreva seu cliente ideal. Qual é a idade? Profissão? Renda?"
│  └─ Objetivo: Criar persona detalhada
│
├─ "Qual é o maior problema/dor que seu cliente tem?"
│  └─ Objetivo: Identificar a dor (base do conteúdo)
│
├─ "Como seu cliente busca solução hoje? (Google, Instagram, indicação?)"
│  └─ Objetivo: Saber em que plataformas estão
│
├─ "Seu cliente já sabe que ele tem este problema?"
│  └─ Objetivo: Nível de consciência (inconsciente, consciente, pronto)
│
└─ "Qual é o ticket médio de uma venda sua?"
   └─ Objetivo: Calcular LTV e ROI realista

┌─────────────────────────────────────────────────────────────────────────────┐
│ BLOCO 3: MERCADO E COMPETIÇÃO (10 min)                                     │
└─────────────────────────────────────────────────────────────────────────────┘

Perguntas:
├─ "Quem são seus 3 maiores concorrentes?"
│  └─ Objetivo: Mapear mercado
│
├─ "O que eles estão fazendo bem que você quer aprender?"
│  └─ Objetivo: Identificar gaps (onde você pode ganhar)
│
├─ "O que eles estão fazendo MAL que você pode explorar?"
│  └─ Objetivo: Encontrar oportunidade única
│
└─ "O que torna seu serviço diferente deles?"
   └─ Objetivo: Confirmar diferencial (ângulo)

┌─────────────────────────────────────────────────────────────────────────────┐
│ BLOCO 4: HISTÓRICO E TENTATIVAS ANTERIORES (10 min)                        │
└─────────────────────────────────────────────────────────────────────────────┘

Perguntas:
├─ "Você já tentou marketing antes? (Ads, conteúdo, influencer?)"
│  └─ Objetivo: Entender experiência + erros
│
├─ "O que funcionou? O que não funcionou?"
│  └─ Objetivo: Não repetir erros, repetir vencedores
│
├─ "Qual foi o melhor resultado que você obteve? Como alcançou?"
│  └─ Objetivo: Entender padrão de sucesso
│
└─ "Por que isso funcionou?"
   └─ Objetivo: Profundidade (mindset + tática)

┌─────────────────────────────────────────────────────────────────────────────┐
│ BLOCO 5: ORÇAMENTO E EXPECTATIVA (10 min)                                  │
└─────────────────────────────────────────────────────────────────────────────┘

Perguntas:
├─ "Qual é seu orçamento total para marketing nos próximos 6 meses?"
│  └─ Objetivo: Desenhar plano viável
│
├─ "Esse orçamento é fixo ou podemos aumentar se funcionar?"
│  └─ Objetivo: Saber se há flexibilidade (importante para escala)
│
├─ "Qual é a meta de leads/vendas que você espera?"
│  └─ Objetivo: Definir KPI (é realista ou sonho?)
│
└─ "Se a gente alcançar essa meta, quanto você fatura?"
   └─ Objetivo: Calcular ROI (vale a pena investir?)

┌─────────────────────────────────────────────────────────────────────────────┐
│ BLOCO 6: PREFERÊNCIAS E RESTRIÇÕES (10 min)                                │
└─────────────────────────────────────────────────────────────────────────────┘

Perguntas:
├─ "Qual é o tom de voz da sua marca? (Formal, descontraído, técnico?)"
│  └─ Objetivo: Guardar tom para Copywriter
│
├─ "Qual plataforma você quer focar? (IG, TikTok, LinkedIn, Email?)"
│  └─ Objetivo: Priorizar esforço
│
├─ "Há algo que você NÃO quer fazer? (Polêmica, certos públicos, etc?)"
│  └─ Objetivo: Evitar problemas
│
└─ "Qual é a melhor forma de comunicar com você? (Email, Slack, WhatsApp?)"
   └─ Objetivo: Comunicação fluida

─────────────────────────────────────────────────────────────────────────────

PÓS-ENTREVISTA: O ESTRATEGISTA FAZ
├─ Resume em documento (Briefing de Cliente)
├─ Identifica o ÂNGULO FORA DA CURVA
├─ Monta apresentação NotebookLM
├─ Envia ao HEAD para validação
└─ Se OK → avança para planejamento estratégico

═══════════════════════════════════════════════════════════════════════════════
PARTE 3: FLUXO COMPLETO (Do Cliente até Primeira Publicação)
═══════════════════════════════════════════════════════════════════════════════

SEMANA 1: IMERSÃO
─────────────────

SEGUNDA (Entrevista)
├─ Estrategista: Coleta dados (60 min com cliente)
├─ Estrategista: Identifica ângulo + escreve Briefing de Cliente
├─ Head: Valida viabilidade (15 min)
└─ Resultado: GREEN LIGHT ou ajuste

TERÇA-QUARTA (Documentação)
├─ Estrategista: Monta apresentação NotebookLM
├─ Head: Revisão final (ortografia, números)
└─ Resultado: Pronto para apresentação

QUINTA (Apresentação)
├─ Estrategista: Apresenta ao cliente (30 min)
├─ Client aprova ou pede ajuste
└─ Resultado: Contrato assinado + pagamento (se necessário)

SEXTA (Kick-off Interno)
├─ Squad reunido (30 min)
├─ Estrategista: Passa briefing para todos
├─ Cada agente entende seu papel
└─ Resultado: GREEN LIGHT para começar

SEMANA 2-3: PLANEJAMENTO
────────────────────────

SEGUNDA (Briefing Estratégico)
├─ Estrategista: Cria SWOT + Funil + Calendário
├─ Copywriter: Contribui tom de voz
├─ Designer: Contribui templates
├─ Traffic Manager: Contribui orçamento
├─ UI Designer: Contribui LP structure
├─ Engajador: Contribui fluxos ManyChat
└─ Head: Valida tudo (conecta?)

TERÇA-QUARTA (Detalhamento)
├─ UI Designer: Começa rascunho LP
├─ Designer: Cria templates A/B/C/D
├─ Copywriter: Prepara tone guidelines
├─ Traffic Manager: Começa setup de campanhas (não ativa ainda)
└─ Publicador: Cria estrutura de Drive

QUINTA (Aprovação Final)
├─ Head: Última revisão (viabilidade OK?)
├─ Estrategista: Prepara apresentação final para cliente
└─ Resultado: Pronto para apresentar

SEXTA (Apresentação ao Cliente)
├─ Estrategista: Apresenta plano 6 meses
├─ Head: Responde objeções técnicas
├─ Cliente aprova + autoriza investimento
└─ Resultado: Contrato assinado + 1º pagamento

SEMANA 4: BUILD (Construir Ativos)
───────────────────────────────────

SEGUNDA (Copy + Design)
├─ Copywriter: Escreve 12 legendas (mês 1)
├─ Designer: Cria 12 carrosseis (mês 1)
├─ Designer: Cria 3 vídeos (15s, 30s, 60s)
└─ Ambos trabalham juntos (sync design + copy)

TERÇA (UI Designer + Landing Page)
├─ UI Designer: Finaliza LP em React/Tailwind
├─ Setup: Formulário, pixels, CTA
├─ Teste: Carregamento rápido? Conversão OK?
└─ Result: LP pronta para deploy

QUARTA (Traffic Manager + Engajador)
├─ Traffic Manager: Configura campanhas (sem ativar)
│  ├─ Públicos definidos
│  ├─ Orçamento alocado
│  ├─ Pixels instalados
│  └─ UTMs criados
├─ Engajador: Setup ManyChat (fluxos testados)
└─ Ambos: Validam com Head

QUINTA (Publicador + Checklist)
├─ Publicador: Estrutura Drive (pastas, nomes)
├─ Publicador: Cria calendário visual
├─ Publicador: Checklist de segurança pronto
├─ Head: Valida TUDO (antes de ir ao ar)
└─ Resultado: 100% pronto para deploy

SEMANA 5: DEPLOY (Ligar os Motores)
────────────────────────────────────

SEGUNDA (Go Live)
├─ UI Designer: LP AO VIVO (monitora 2h)
├─ Traffic Manager: Campanhas ATIVADAS (30% orçamento teste)
├─ Publicador: Primeiros 3 posts publicados
├─ Engajador: ManyChat disparando (triagem automática)
└─ Head: Monitora alertas (algo quebrou?)

TERÇA-SEXTA (Monitoramento 24/7)
├─ Traffic Manager: Daily optimization
│  ├─ Monitora CTR, CPC, CPA
│  ├─ Pausa criativos ruins
│  └─ Escala vencedores (se houver)
│
├─ Publicador: Acompanha 24-48h
│  ├─ Responde comentários iniciais
│  ├─ Monitora reach/impressions
│  └─ Documenta performance
│
├─ Engajador: Qualifica leads manualmente
│  ├─ Triage em DM
│  ├─ Coleta dores
│  └─ Documenta interações
│
└─ Head: Standup diário (5 min, qualquer bloqueio?)

SEMANA 6+: OPERAÇÃO CONTÍNUA
────────────────────────────

DIÁRIO:
├─ Traffic Manager: Otimiza campanhas (30 min)
├─ Engajador: Qualifica leads (60 min)
├─ Publicador: Monitora performance (15 min)
└─ Qualquer problema? → Escalação ao Head

SEMANAL:
├─ Publicador: Publica 2-3 novos conteúdos
├─ Engajador: Relatório de dores/objeções (para Copywriter/Estrategista)
├─ Head: Standup squad (15 min, tudo OK?)
└─ Copywriter: Cria variações para A/B test

MENSAL:
├─ Traffic Manager: Relatório consolidado (ROI, CPL, leads)
├─ Publicador: Analytics por plataforma
├─ Estrategista: Narrativa "próximos passos"
├─ Apresentação ao cliente (o que saiu, dados, insights)
└─ Decision: Escala, mantém ou pivota?

CONCLUSÃO: COMO VOCÊ VAI USAR A ESTRUTURA

1. CLIENTE CHEGA
   └─ Entrevista com Estrategista (BRIEFING DE CLIENTE)
      └─ Identifica ÂNGULO FORA DA CURVA
         └─ Apresenta ao cliente (NotebookLM)
            └─ Cliente aprova
               └─ Squad recebe BRIEFING ESTRATÉGICO
                  └─ Cada agente sabe exatamente o que fazer
                     └─ 6 meses de execução + mensuração
                        └─ Cliente vê resultado transparente

2. DADOS FLUEM
   └─ Traffic Manager → números de performance
      └─ Publicador → alcance e engajamento
         └─ Engajador → dores e objeções reais
            └─ Tudo alimenta BRIEFING ESTRATÉGICO (ajustado mensalmente)
               └─ Copywriter cria conteúdo que REALMENTE funciona
                  └─ Cycle de melhoria contínua

3. DESIGN SOCIAL COM TEMPLATES CANVA (NOVO!)
   └─ Designer Social recebe copy + imagens
      └─ Seleciona template apropriado (A/B/C/D)
         └─ Customiza imagem + texto no Canva
            └─ Exporta PNG
               └─ Publicador sobe no Instagram

   EXEMPLO: Carrossel sobre "IA na produtividade"
   ├─ Slide 1 → Template A (Gancho) → Full-bleed imagem
   ├─ Slide 2 → Template B (Conteúdo) → 50% imagem + texto
   ├─ Slide 3 → Template B (Conteúdo) → prova social
   ├─ Slide 4 → Template C (Educação) → 3 tips numerados
   └─ Slide 5 → Template D (CTA) → botão amarelo "Clique"

   Tempo por slide: ~5 min (trocar imagem + texto)
   Total carrossel 5 slides: ~30 min
   Vs. design do zero: ~4-5h

4. FERRAMENTA: NotebookLM
   └─ Cliente vê TUDO (transparência)
      └─ Squad vê TUDO (alinhamento)
         └─ Head controla TUDO (governança)
            └─ Dados estruturados → decisões rápidas
