Guia de Implementação Canva

1. Como Usar os Templates no Canva

Cada template deve ser criado como um "frame" reutilizável no Canva. Aqui está o passo a passo:

  • Criar novo design: 1080 × 1350 px (Instagram Post)
  • Adicionar background: Cor sólida, gradiente ou imagem
  • Adicionar elemento: Imagem, texto, botão conforme template
  • Salvar como template: Nome: "Template A - Gancho", etc
  • Duplicar para variações: Claro/escuro, com/sem logo

2. Sistema de Cores

Paleta Principal:

  • Primário: #FFD700 (Amarelo vibrante)
  • Dark: #000000 (Preto puro)
  • Light: #FFFFFF (Branco puro)
  • Gray Light: #F5F5F5 (Off-white)
  • Gray Dark: #1A1A1A (Cinza escuro)

Uso: Amarelo apenas para destaque (CTA, números, keywords). Preto e branco para contrast.

3. Tipografia

  • Display Font: Courier New (monospace, geométrica)
  • Body Font: Courier New ou Verdana
  • Headline: 28-40px, bold, letter-spacing -1px a -2px
  • Body Text: 14-18px, regular, line-height 1.4-1.6
  • Small Text: 12-14px, regular, line-height 1.5

4. Grid & Spacing

  • Safe Area: Deixe 40px de padding em todas as bordas (evita corte em mobile)
  • Espaço entre elementos: Múltiplos de 10px (10, 20, 30, 40)
  • Alinhamento: Use grid de 10px para alinhar elementos
  • Respiro Visual: Não aglomere. Prefira espaço em branco a densidade

5. Checklist de Qualidade (Antes de Usar)

  • ☐ Dimensões corretas (1080 × 1350px)?
  • ☐ Texto legível (16px mínimo para body)?
  • ☐ Contraste OK (WCAG AA, mínimo 4.5:1 ratio)?
  • ☐ Sem safe area violations (40px padding)?
  • ☐ Cores alinhadas com brand (#FFD700 destaque)?
  • ☐ Tipografia consistente (Courier New)?
  • ☐ Sem poluição visual (menos é mais)?
  • ☐ CTA claro e acionável (Template D)?

6. Criação de Variações

Para cada template, crie pelo menos estas variações:

  • Versão 1: Fundo claro + com logo branded
  • Versão 2: Fundo escuro + com logo branded
  • Versão 3: Fundo claro + sem logo (para clientes que customizam)
  • Versão 4: Fundo escuro + sem logo (para clientes que customizam)

Total: 4 templates × 4 variações = 16 frames no Canva.

7. Fluxo de Trabalho (Designer Social)

  1. Receber brief do Estrategista (qual template? Qual mensagem?)
  2. Escolher template correspondente (A/B/C/D)
  3. Customizar: imagem, texto, cores (se necessário)
  4. Validar: checklist de qualidade acima
  5. Enviar ao Publicador para agendamento
