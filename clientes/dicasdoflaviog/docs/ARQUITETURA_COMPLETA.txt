2. NOVA: AGENTE UI DESIGNER (Especialista em Conversão Visual)
════════════════════════════════════════════════════════════════

PAPEL:
└─ Converte estrutura + copy em design/componentes visuais
└─ Executa em React + Tailwind CSS
└─ Responsável por Performance, UX, Acessibilidade
└─ Domina 4 estilos: Minimalista, Moderno, Luxo, Extravagante

COMPETÊNCIAS PRINCIPAIS:

1. USABILIDADE & UX (Mobile-First)
   └─ 44px+ botões, espaçamento correto, zero carga cognitiva

2. PERFORMANCE (Otimização)
   └─ WebP/SVG, lazy loading, LCP < 2.5s

3. MICROANIMAÇÕES (Vida sem peso)
   └─ Framer Motion / Tailwind Animate (300-500ms)

4. TIPOGRAFIA ELITE (Comunicação)
   └─ Fontes que transmitem autoridade + legibilidade

5. VISUAL STORYTELLING (Narrativa)
   └─ Cada seção prepara a próxima (fluxo emocional)

6. ACESSIBILIDADE (Padrão de Excelência)
   └─ ARIA, Alt text, Semântica HTML, Contraste WCAG AA

ENTREGA:
└─ Componentes React prontos (Tailwind CSS)
└─ Design Tokens documentados (cores, spacing, fonts)
└─ Pronto para produção ou passar para DEV

8. EXEMPLO DE FLUXO COMPLETO
════════════════════════════════════════════════════════════════

CENÁRIO: Criar carrossel sobre "IA na produtividade"

DIA 1 — SEGUNDA
────────────────
10:00 - Sol envia brief:
"Carrossel 6 slides sobre como IA automatiza tarefas repetitivas.
Público: Donos de pequenos negócios. Tom: Prático, não teórico."

10:15 - HEAD aprova viabilidade
"Viável. Orçamento: 2h Estrategista + 3h Copy + 4h Design. Prossiga."

10:30 - ESTRATEGISTA trabalha
Cria structure:
- Slide 1: Gancho (problema: "Você gasta 8h/semana em email")
- Slide 2: Contexto (estatística: 70% do tempo é tarefas repetidas)
- Slide 3: Solução (IA automatiza essas tarefas)
- Slide 4: Exemplo prático (email + agenda)
- Slide 5: Prova social (clientes usando)
- Slide 6: CTA (comece agora)

11:00 - HEAD revisa estrutura
"Aprovado. Passa para Copy e Design."

13:00 - COPYWRITER escreve
Cria copy para cada slide, captions

14:00 - UI DESIGNER começa mockup
(Pode trabalhar paralelo ao copywriter)

17:00 - HEAD revisa copy + design
"Copy: Aprovado. Design: Rejeitado (cor de fundo muito clara).
Copywriter: refaz. Designer: ajusta fundo + cor de CTA."

18:30 - COPYWRITER e UI DESIGNER entregam versão 2

19:00 - HEAD aprova ambos
"Go! Passa para publicação."

DIA 2 — TERÇA
─────────────
08:00 - PUBLICADOR formata para Instagram
Caption: "Você gasta quanto tempo em tarefas que um bot poderia fazer?
[Copy do slide 1]
[Link para mais]"

Hashtags: #IA #Automação #Produtividade #Small Business

Alt text: "6 slides sobre automação com IA em pequenos negócios"

Horário: Terça 12:30 (pico de atividade)

09:00 - HEAD revisa
"Caption pode ser mais direto. Refaz."

10:00 - PUBLICADOR entrega final
"Pronto pra publicar"

10:15 - PUBLICAÇÃO INSTAGRAM

10:30+ - ENGAJADOR monitora respostas
Responde comentários, redireciona DMs
HEAD monitora se ton está correto

9. DESIGN SYSTEM: TEMPLATES CANVA A/B/C/D
════════════════════════════════════════════════════════════════

NOVO (Março 2026): 4 Templates prontos para reutilizar em todos os clientes.

TEMPLATE A — GANCHO / ABERTURA
└─ Estrutura: Imagem full-bleed (1080×1350) + Texto mínimo (bottom)
└─ Uso: Sempre o PRIMEIRO slide do carrossel
└─ Propósito: Parar o scroll, gerar curiosidade
└─ Variações: Light, Dark, Com Logo, Sem Logo
└─ Checklist:
   ├─ Imagem otimizada (WebP, 1080p+)
   ├─ Texto branco/preto com bom contraste
   ├─ Safe area: 40px padding
   └─ Headline: 36-40px, bold, centered bottom

TEMPLATE B — CONTEÚDO / HUMANIZAÇÃO
└─ Estrutura: 50% esquerda (imagem) + 50% direita (texto)
└─ Uso: Slides 2-4 do carrossel, prova social
└─ Propósito: Educação + confiança
└─ Variações: Light, Dark, Com Logo, Imagem Direita
└─ Checklist:
   ├─ Proporção 50/50 respeitada
   ├─ Headline 24px, body 14px
   ├─ Imagem não pixelada
   └─ Respiro visual entre elementos

TEMPLATE C — EDUCAÇÃO / LISTAS
└─ Estrutura: Fundo sólido (preto ou cinza) + 3-5 items numerados
└─ Uso: Listas, rankings, step-by-step
└─ Propósito: Informação estruturada
└─ Variações: Dark, Light, Com Logo, 3-items, 5-items
└─ Checklist:
   ├─ Números em #FFD700 (amarelo)
   ├─ Título 20px, descrição 14px
   ├─ Espaçamento 200px entre items
   └─ Fundo sem textura (sólido)

TEMPLATE D — CTA / FECHAMENTO
└─ Estrutura: Fundo escuro + Headline grande + Botão destaque
└─ Uso: SEMPRE o ÚLTIMO slide
└─ Propósito: Conversão
└─ Variações: Dark, Light, Com Circulo Decorativo
└─ Checklist:
   ├─ Headline 32px, bold, white
   ├─ Botão #FFD700, 200×50px
   ├─ CTA clara ("Clique aqui", "Baixar agora")
   └─ Safe areas respeitadas

COMO USAR (Designer Social):
1. Abra link Canva compartilhado (CANVA_LINK.txt)
2. Selecione template apropriado
3. Clique na imagem → Replace → upload sua foto
4. Edite textos (headline/body)
5. Valide: cores, tipografia, safe areas
6. Download PNG
7. Salve em /outputs/carrosseis/[tema]-slide-[n].png

ESPECIFICAÇÕES TÉCNICAS:
├─ Dimensões: 1080 × 1350px (Instagram)
├─ Cores: #FFD700 (amarelo) + #000000 + #FFFFFF
├─ Tipografia: Courier New (display + body)
├─ Safe area: 40px padding mínimo
├─ Imagens: WebP, 1080p ou maior
└─ Exporte: PNG (ou WebP se o Editor suportar)
