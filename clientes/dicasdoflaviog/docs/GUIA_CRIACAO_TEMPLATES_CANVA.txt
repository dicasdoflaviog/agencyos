═══════════════════════════════════════════════════════════════════════════════
PARTE 1: PREPARAÇÃO INICIAL (15 min)
═══════════════════════════════════════════════════════════════════════════════

PASSO 1: Criar Novo Projeto no Canva
────────────────────────────────────
1. Acesse canva.com (já logado?)
2. Clique em "Criar Design"
3. Selecione "Instagram Post" (1080 × 1350 px)
4. Nomeie: "Sol Templates A/B/C/D - [DATA]"
5. Clique em "Brand Kit" (adicione as cores)

PASSO 2: Setup Brand Kit (Cores + Fontes)
──────────────────────────────────────────
No Canva, vá para Brand Kit e adicione:

CORES:
├─ Primário: #FFD700 (Amarelo vibrante) — USE PARA DESTAQUE
├─ Dark: #000000 (Preto puro) — FUNDO OU TEXTO ESCURO
├─ Light: #FFFFFF (Branco puro) — FUNDO OU TEXTO CLARO
├─ Gray Light: #F5F5F5 (Off-white) — FUNDO ALTERNATIVO
└─ Gray Dark: #1A1A1A (Cinza escuro) — FUNDO ALTERNATIVO

TIPOGRAFIA:
├─ Display: Courier New (ou Courier) — HEADLINES
├─ Body: Verdana ou Courier New — TEXTO CORPO
└─ Tamanho padrão:
   ├─ Headline: 32-40px, Bold, Letter-spacing -2px
   ├─ Body: 16-18px, Regular, Line-height 1.5
   └─ Small: 12-14px, Regular, Line-height 1.5

RESULTADO: Brand Kit salvo (1-2 min por nova página)

═══════════════════════════════════════════════════════════════════════════════
PARTE 2: CRIAR TEMPLATE A (Gancho / Abertura) — 20 min
═══════════════════════════════════════════════════════════════════════════════

TEMPLATE A = Imagem Full-Bleed + Texto Mínimo

PASSO 1: Criar novo frame
────────────────────────
1. Clique em "+" → "Frame"
2. Dimensões: 1080 × 1350 (já preenchidas)
3. Nomeie: "A - Gancho (Light)"

PASSO 2: Adicionar fundo (imagem exemplo)
──────────────────────────────────────────
1. Clique em "Uploads" ou "Photos" → escolha imagem
2. Faça upload ou selecione do Canva Stock
3. Resize para ocupar 100% do frame (Image Fill)
4. Sem padding (full-bleed mesmo)

NOTA: No uso real, Designer Social vai trocar a imagem.
Agora é só exemplo visual.

PASSO 3: Adicionar overlay gradiente (opcional)
───────────────────────────────────────────────
1. Clique em "+" → "Shape" → "Retângulo"
2. Tamanho: 1080 × 1350 (preenche tudo)
3. Cor: Preto com 60% opacity
4. Posição: Sobrepõe imagem
5. Alinha no topo (começa do top)

RESULTADO: Imagem escurece levemente no bottom (mais legível)

PASSO 4: Adicionar texto (Headline)
───────────────────────────────────
1. Clique em "+" → "Texto"
2. Escreva: "Sua Rotina em Metade do Tempo" (exemplo)
3. Fonte: Courier New, 40px, Bold, Branco
4. Alinhamento: Center
5. Posição: Bottom (40px acima da borda)
6. Largura: 900px (deixa respiro nas laterais)

RESULTADO: Texto branco legível no bottom da imagem

PASSO 5: Adicionar logo (opcional)
──────────────────────────────────
1. Clique em "+" → "Uploads" → logo dicasdoflaviog
2. Resize: 60 × 60px
3. Posição: Bottom-left corner (20px margins)
4. Opacity: 80% (não compete com texto)

RESULTADO: Logo presente mas discreto

PASSO 6: Duplicar para "A - Gancho (Dark)"
──────────────────────────────────────────
1. Frame inteiro → clique direito → "Duplicate"
2. Nomeie: "A - Gancho (Dark)"
3. Mude fundo para imagem escura (ou use overlay 80% preto)
4. Mantém texto e logo iguais

RESULTADO: 2 variações de Template A criadas ✅

═══════════════════════════════════════════════════════════════════════════════
PARTE 3: CRIAR TEMPLATE B (Conteúdo + Humanização) — 20 min
═══════════════════════════════════════════════════════════════════════════════

TEMPLATE B = 50% Imagem (esquerda) + 50% Texto (direita)

PASSO 1: Criar novo frame
────────────────────────
1. Clique em "+" → "Frame"
2. Nomeie: "B - Conteúdo (Light)"

PASSO 2: Adicionar imagem (esquerda 50%)
────────────────────────────────────────
1. Clique em "+" → "Uploads" → escolha imagem
2. Faça resize: 540px wide × 1350px tall (metade esquerda)
3. Posição: X=0, Y=0 (canto superior esquerdo)
4. Sem padding

PASSO 3: Adicionar fundo texto (direita 50%)
─────────────────────────────────────────────
1. Clique em "+" → "Shape" → "Retângulo"
2. Cor: Branco (#FFFFFF)
3. Tamanho: 540px wide × 1350px tall
4. Posição: X=540, Y=0 (lado direito)

RESULTADO: Tela dividida 50/50

PASSO 4: Adicionar headline (direita)
─────────────────────────────────────
1. Clique em "+" → "Texto"
2. Escreva: "Como Economizar 10h/Semana" (exemplo)
3. Fonte: Courier New, 24px, Bold, Preto
4. Posição: Direita, padding 30px
5. Y = 80px (20% do topo)

PASSO 5: Adicionar body text (direita)
──────────────────────────────────────
1. Clique em "+" → "Texto"
2. Escreva: "5 freelancers aprenderam a estrutura..." (max 100 chars)
3. Fonte: Courier New, 14px, Regular, Preto (opacity 70%)
4. Posição: Direita, padding 30px
5. Y = 150px (abaixo do headline)

RESULTADO: Imagem left + texto right harmonizados

PASSO 6: Duplicar para variações
───────────────────────────────
1. Frame → clique direito → Duplicate 3x
2. Variação 1: "B - Conteúdo (Dark)" — troca fundo direita para #F5F5F5
3. Variação 2: "B - Conteúdo (Com Logo)" — adiciona logo 40×40px no top-right
4. Variação 3: "B - Conteúdo (Imagem Direita)" — inverte: imagem direita, texto esquerda

RESULTADO: 4 variações de Template B criadas ✅

PARTE 8: TROUBLESHOOTING (Erros Comuns)
═══════════════════════════════════════════════════════════════════════════════

PROBLEMA: "Texto ficou cortado nas laterais"
SOLUÇÃO: Adicione 40px padding em todos lados. Canva pode cortar se estiver no extremo.

PROBLEMA: "Imagem ficou pixelada"
SOLUÇÃO: Upload foto em 1080p ou maior. Canva comprime, mas precisa qualidade base.

PROBLEMA: "Cores não bateram com brand"
SOLUÇÃO: Use Brand Kit. Não insira cores manualmente (#FFD700). Pega do kit.

PROBLEMA: "Alinhamento desigual entre frames"
SOLUÇÃO: Use "Align" tool (Canva). Seleciona 2 elementos → Align Center/Left/etc.

PROBLEMA: "Esqueci de salvar"
SOLUÇÃO: Canva auto-save. Mas clique em "Download" quando terminar pra ter backup.
