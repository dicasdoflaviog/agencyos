PARTE 2: SETUP DE CADA PLATAFORMA
════════════════════════════════════════════════════════════════════════════════

2.1 NOTEBOOKLM (IA que entende contexto)
─────────────────────────────────────────

Vai para: notebooklm.google.com
Cria: novo notebook "dicasdoflaviog 2026-S1"

Upload tudo que tiver:
├─ Todos os briefs
├─ Todos os copies anteriores
├─ Histórico de performance
├─ Brand guidelines
├─ Diretrizes de agentes
└─ Este guia (INTEGRACAO_DRIVE_PLATAFORMAS.txt)

HEAD usa assim:
├─ Pergunta: "Este copy 'Descubra os 7 segredos' está alinhado?"
├─ NotebookLM responde: "Não. 'Segredos' é proibido. Use 'pontos'."
└─ HEAD aprova/rejeita com confiança

2.2 MANYCHAT (Automação de comentários)
────────────────────────────────────────

Vai para: manychat.com
Conecta: sua conta Instagram (@dicasdoflaviog)

Cria 2 fluxos:

FLUXO 1 — Resposta Automática
   Trigger: Novo comentário
   Ação: "Oi! 👋 Qual sua maior dúvida sobre [tema]?"
   Resultado: Capta nome + resposta

FLUXO 2 — Qualificação
   Se responder "preço"/"custo" → Lead Hot
   Se responder "dúvida"/"como" → Lead Warm
   Se responder "legal"/"top" → Lead Cold
   Resultado: Envia para Make/Sheets

2.3 META + INSTAGRAM
────────────────────

Meta (business.facebook.com):
├─ Cria Business Manager
├─ Conecta: Ad Account, Página FB, Conta IG
├─ Cria: Pixel (rastreio de cliques)
├─ Configura: Eventos (Lead, Purchase, etc)
├─ Gera: API Token (salva em Drive: 07_REFERENCE/)

Instagram (@dicasdoflaviog):
├─ Converte para conta profissional
├─ Conecta ao Meta Business Manager
├─ Publica posts via Meta Business Suite (agendado)
├─ Monitora Insights (alcance, engajamento)
└─ Dados fluem para Dashboard

2.4 MAKE (Automação de fluxos)
───────────────────────────────

Vai para: make.com
Cria 3 fluxos:

FLUXO 1 — Leads ManyChat → Google Sheets
   Trigger: ManyChat capta novo lead
   Ação 1: Google Sheets adiciona linha
   Ação 2: Slack notifica "@engajador Novo lead!"
   
FLUXO 2 — Performance Meta → Google Sheets
   Trigger: Todo dia às 23h
   Ação: Meta API puxa dados (impressões, clicks, CPL, leads)
   Resultado: Google Sheets aba "Performance Meta" atualiza

FLUXO 3 — Alerta CPL Alto
   Trigger: Se CPL > R$ 200
   Ação: Slack avisa "🚨 CPL alto! Revisar criativo."
   Resultado: TRAFFIC MANAGER age rápido

PARTE 4: FLUXO COMPLETO DE UM JOB
════════════════════════════════════════════════════════════════════════════════

SEGUNDA — Entrevista (Estrategista + você)
───────────────────────────────────────────
Estrategista: "Qual é seu maior problema com marketing?"
Você: "Não consigo escalar. Gasto muito em tráfego, baixa conversão."
   ↓
Estrategista documenta em Google Docs
   ↓
Upload tudo no NotebookLM
   ↓
HEAD acessa NotebookLM, entende contexto

TERÇA — Briefing (Estrategista + HEAD)
───────────────────────────────────────
Estrategista cria: "BRIEF — Automação Email"
   ├─ Público: Freelancers
   ├─ Ângulo: "Email não é trabalho. É desperdício."
   ├─ Estrutura: 5 slides (A/B/C/D)
   └─ Tom: Direto, anti-guru

Salva em: Drive → 01_CLIENTES/dicasdoflaviog/Campanhas/2026-S1/
   ↓
HEAD lê em NotebookLM: "Ângulo está bom?"
HEAD aprova ou retorna feedback

QUARTA — Produção (Copywriter + Designer)
──────────────────────────────────────────
Copywriter:
   ├─ Lê: Brief + NotebookLM
   ├─ Escreve copy (5 slides)
   ├─ Salva: Google Docs no Drive
   ├─ HEAD revisa (ortografia, tom)
   └─ ✅ Aprovado

Designer Social:
   ├─ Lê: Copy + Brief
   ├─ Abre: Templates Canva A/B/C/D
   ├─ Customiza: fontes/cores dicasdoflaviog + imagens reais
   ├─ Exporta: 5 PNGs (1080×1350)
   ├─ Salva: Drive → Projeto 1/Carrosseis/
   ├─ HEAD revisa (contraste, safe areas)
   └─ ✅ Aprovado

QUINTA — Publicação (Publicador)
─────────────────────────────────
Publicador:
   ├─ Abre: Drive → Projeto 1/Carrosseis/
   ├─ Copia: 5 PNGs
   ├─ Vai a: Meta Business Suite (business.facebook.com)
   ├─ Clica: Compose Post → Instagram Feed
   ├─ Upload: 5 PNGs (carrossel)
   ├─ Cola: Caption do Copywriter
   ├─ Adiciona: Hashtags + Alt text
   ├─ Agenda: 14h (melhor horário)
   └─ Schedule

   ↓
Meta Pixel rastreia cliques
   ↓
ManyChat capta comentários

SEGUNDA PRÓXIMA — Monitoramento
────────────────────────────────
Make fluxos rodam:

FLUXO 1: ManyChat → Google Sheets (leads capturados)
   Nomes dos leads
   Perguntas deles
   Classificação (Hot/Warm/Cold)

FLUXO 2: Meta → Google Sheets (performance)
   Impressões
   Clicks
   CPM (custo por mil)
   CPL (custo por lead)
   Total gasto

FLUXO 3: Se CPL > 200, alerta para TRAFFIC MANAGER
   "🚨 CPL alto! Revisar criativo."

Resultado:
   ├─ Dashboard atualiza em tempo real
   ├─ HEAD vê performance
   ├─ Estrategista analisa em NotebookLM
   └─ Próximo brief ataca padrão vencedor
